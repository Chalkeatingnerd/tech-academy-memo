## 1. 문제를 논리적으로 분해

![](./task_breakdown.png)

큰 task를 작은 하위 task로 분해

### 예시

문제: 배송비 포함 가격 계산

상세 내용: 세금 별도 가격을 기준으로 소비세 포함 가격과 배송비를 합한 '배송비 포함 가격'을 계산하고 표시하는 코드를 작성하시오. 부가세 포함 가격이 2000이상인 경우는 무료 배송, 2000엔 미만인 경우 배송비는 350엔, 소비세율은 10%

이대로 소스 코드 작성 난해

### 예시 문제 논리 분해

1. 세금 별도 가격 입력
2. 세금 포함 가격 계산
3. 배송비 계산
    부가세 포함 가격이 2000엔 이상: 무료 배송
    부가세 포함 가격이 2000엔 미만: 배송비 350엔
4. '배송비 포함 가격은 XXX엔입니다' 라고 표시

## 2. 컴퓨터가 처리할 수 있는 형식으로 표현

데이터 수신 → 데이터 가공 → 데이터 출력

- 처리 데이터 변수 구분

1. 세금 별도 가격(input)
2. 세금 포함 가격: 세금 별도 가격 x 1.1
3. 배송비 계산
    세금 포함 가격 >=2000: 배송비 0
    그외: 배송비 350
4. 배송비 포함 가격: 세금 포함 가격 + 배송비
